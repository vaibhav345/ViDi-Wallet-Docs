## 6. Data Security

We implement industry-standard security measures to protect your information:

- **Encryption:** All data transmitted between your device and our servers is encrypted using TLS/SSL
- **Database Security:** Your financial data is stored in encrypted SQLite databases on your device
- **Cloud Security:** Cloud-synced data (Premium feature) is encrypted at rest and in transit
- **Authentication:** Firebase Authentication with secure password hashing and session management
- **Payment Security:** Stripe handles all payment processing with PCI-DSS compliance; we never store full card numbers
- **Access Controls:** Limited employee access to personal data on a need-to-know basis

Despite our efforts, no security system is 100% secure. We cannot guarantee absolute security of your information.

## 11. Analytics and Performance Monitoring

### 11.1 Firebase Analytics

We use Firebase Analytics to understand how users interact with the App. Firebase Analytics collects:
- App usage events (feature usage, screen views, button clicks)
- Device information (OS version, device model, language)
- User demographics (country, age range, gender - if provided)

**Privacy Controls:**
- You can opt-out of Firebase Analytics in Settings > Privacy > "Share Usage Analytics"
- When opted out, we collect only crash data for app stability purposes

### 11.2 Firebase Crashlytics

We use Firebase Crashlytics to detect and diagnose app crashes. Crashlytics collects:
- Crash reports (stack traces, device state at time of crash)
- Device information (OS version, device model, free memory)
- Custom logs (non-personal technical information)

**Privacy Controls:**
- You can opt-out of Crashlytics in Settings > Privacy > "Send Crash Reports"
- Opting out may limit our ability to fix bugs affecting your experience

### 11.3 Firebase Performance Monitoring

We use Firebase Performance Monitoring to track app performance metrics:
- Page load times, network latency, API response times
- Device performance (CPU usage, memory consumption)
- User experience metrics (app startup time, screen rendering)

**Privacy Controls:**
- You can opt-out of Performance Monitoring in Settings > Privacy > "Performance Monitoring"
- Performance data is aggregated and does not include personal identifiers

### 11.4 Data Anonymization

All analytics data is anonymized and aggregated before being used for insights. We do not use analytics to identify individual users.
